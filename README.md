📚 LLM-Powered Book Recommendation System

An intelligent book recommendation system built using LLMs, LangChain, ChromaDB, and Gradio. This project leverages modern AI techniques to provide personalized book suggestions based on user queries and semantic similarity.

🚀 Features

🔍 Semantic book search using embeddings

🤖 LLM-powered recommendations

📊 Emotion-aware book filtering (if dataset supports it)

⚡ Fast vector search with ChromaDB

🌐 Interactive UI with Gradio

🧠 Local embeddings using Ollama

📚 Hugging Face models integration

🛠️ Tech Stack

Python

LangChain

ChromaDB

Gradio

Ollama (for local embeddings)

Hugging Face Transformers

Pandas / NumPy
